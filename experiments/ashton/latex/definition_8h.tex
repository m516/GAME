Reads a fragment from file {\itshape file\+Name} starting with line {\itshape start\+Line} and ending with line {\itshape end\+Line}. The result is returned as a string via {\itshape result}. The function returns T\+R\+UE if successful and F\+A\+L\+SE in case of an error.

Reads a fragment of code from file {\itshape file\+Name} starting at line {\itshape start\+Line} and ending at line {\itshape end\+Line} (inclusive). The fragment is stored in {\itshape result}. If F\+A\+L\+SE is returned the code fragment could not be found.

The file is scanned for a opening bracket (\textquotesingle{}\{\textquotesingle{}) from {\itshape start\+Line} onward The line actually containing the bracket is returned via start\+Line. The file is scanned for a closing bracket (\textquotesingle{}\}\textquotesingle{}) from {\itshape end\+Line} backward. The line actually containing the bracket is returned via end\+Line. Note that for V\+H\+DL code the bracket search is not done.
