\doxysubsection{Detailed Description}
The interface to the object model provided by the X\+ML parser library. 

To start using this library one calls \mbox{\hyperlink{doxmlintf_8h_addc96d6d7f1823dbe573dd703865568c}{create\+Object\+Model()}} and then uses the returned \mbox{\hyperlink{class_i_doxygen}{I\+Doxygen}} interface to read doxygen generated X\+ML output and navigate through the information contained in it.
