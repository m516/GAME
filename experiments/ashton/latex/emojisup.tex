The \href{http://www.unicode.org/}{\texttt{ Unicode consortium}} has defined a set of \href{https://en.wikipedia.org/wiki/Emoji}{\texttt{ emoji}} with the corresponding unicode sequences. \mbox{\hyperlink{class_doxygen}{Doxygen}} supports the subset of emoji characters as used by Git\+Hub (based on the list \href{https://api.github.com/emojis}{\texttt{ https\+://api.\+github.\+com/emojis}}). An emoji is created using the \textbackslash{}emoji command. For example {\ttfamily \textbackslash{}emoji smile} (or {\ttfamily \textbackslash{}emoji \+:smile\+:}) both produce \doxygenemoji{\+:smile\+:}{smile}.\hypertarget{emojisup_emojirep}{}\doxysection{Representation}\label{emojisup_emojirep}
For the different doxygen output types there is an output defined\+:
\begin{DoxyItemize}
\item Unicode code sequence, the actual representation is depending on the possibilities of the fonts loaded\+:
\begin{DoxyItemize}
\item H\+T\+ML
\item Doc\+Book
\item R\+TF, converted to U\+T\+F-\/16 representation.
\end{DoxyItemize}
\item \mbox{\hyperlink{class_image}{Image}}
\begin{DoxyItemize}
\item \textbackslash{}\+La\+TeX, in case the image can be found (see \mbox{\hyperlink{emojisup_emojiimage}{Emoji image retrieval}}) otherwise the plain emoji text (i.\+e. {\ttfamily \+:$<$text$>$\+:}) is displayed
\end{DoxyItemize}
\item plain emoji text (i.\+e. {\ttfamily \+:$<$text$>$\+:})
\begin{DoxyItemize}
\item man
\item perl
\end{DoxyItemize}
\item For X\+ML there is a dedicated {\ttfamily $<$emoji$>$} tag with name and unicode attributes.
\end{DoxyItemize}\hypertarget{emojisup_emojiimage}{}\doxysection{Emoji image retrieval}\label{emojisup_emojiimage}

 When invoking it with the {\ttfamily -\/d image\+\_\+dir} option the images will by downloaded in the {\ttfamily image\+\_\+dir} directory. By means of the doxygen configuration parameter \mbox{\hyperlink{config_cfg_latex_emoji_directory}{L\+A\+T\+E\+X\+\_\+\+E\+M\+O\+J\+I\+\_\+\+D\+I\+R\+E\+C\+T\+O\+RY}} the requested directory can be selected.

For convenience a zip with the result of running the script can also be downloaded from \href{http://www.doxygen.nl/dl/github_emojis.zip}{\texttt{ http\+://www.\+doxygen.\+nl/dl/github\+\_\+emojis.\+zip}}

For an overview of the supported emoji one can issue the command\+:~\newline
 {\ttfamily doxygen -\/f emoji $<$output\+File\+Name$>$}
